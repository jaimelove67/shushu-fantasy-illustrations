# 幻想泡泡与构图模式

## 云泡泡规则

- 主云是白色、圆鼓鼓、不规则的手绘云团。
- 轮廓用黑色或深棕色粗线，允许有重复描线和断线。
- 用 2-3 个小圆泡泡从鼠鼠头顶连到主云。
- 主云内放一个中心幻想，不要塞满。
- 云里可以是贴纸、照片感物件、手绘道具或另一个小鼠鼠。

## 常用构图

### 真实场景 + 一个大幻想

适合公园、实验室、教室、办公室、街景。鼠鼠在前景下方，幻想云在右上或上方，云里是它想要的身份、对象或奖励。

### 白底模板

适合表情包底图。鼠鼠在左下或下方，右上是大云泡泡。背景保持纯白，云里放一个金币、证书、恋爱小剧场、学士帽鼠鼠等。

### 多鼠鼠同场

适合会议、班级、实验室、合照。画面中出现 2-4 只鼠鼠，每只可以有不同幻想云。注意不要让云泡泡互相遮挡。

### 身份幻想

现实鼠鼠穿普通白身体，幻想云里穿毕业服、安全帽、工装、老板西装或摄影师装备。现实和幻想形成反差。

### 恋爱幻想

云里放两只害羞鼠鼠、玫瑰、腮红、短字“嘿嘿”“嘻嘻OAO”。不要画成正式情侣插画，要像脑内小剧场。

### 暴富幻想

云里放金币、token、工资条、奖杯或“发财”。鼠鼠现实里小小站着，幻想里物件很大。

## 字词池

默认不使用文字。确实需要时，只少量使用情绪词：

- 嘻嘻
- 嘿嘿
- OAO
- 好耶

文字要像梗图点缀，不要变成说明。

## 名人/身份幻想规则

名人、职业、身份或成功幻想必须用视觉符号表达，不用文字直说。

- 赛车冠军：用卡丁车现实场景、F1 风赛车、44 号、头盔、领奖台、香槟、速度线表达。
- 毕业：用学士帽、毕业袍、证书、校园台阶表达。
- 工程师：用安全帽、工牌、蓝图、控制室表达。
- 摄影师：用相机、取景框、大片姿势表达。
- 暴富：用金币、奖杯、工资条、巨大硬币表达。
- 恋爱：用玫瑰、腮红、两只鼠鼠靠近表达。

避免写“我是某某”“我要发财”“我毕业了”“我是冠军”。如果文字删掉后笑点不成立，说明视觉设计不够明确，要重画云里的幻想内容。

## 避免复刻

不要默认复刻 `assets/examples/` 的具体构图：

- 公园草地 + 相机鼠鼠 + 恋爱幻想云。
- 白底左下大牙鼠鼠 + 右上 token 云。
- 实验室汇报场景 + 多个鼠鼠幻想云。

可以继承“真实场景 + 鼠鼠贴纸 + 幻想云”的结构，但每次要换幻想对象、道具、位置关系或叙事角度。
